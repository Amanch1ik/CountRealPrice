# SoftCost — Система автоматизированного расчета себестоимости ПО

**SoftCost** — это современное веб-приложение, разработанное для автоматизации процесса формирования себестоимости программных продуктов. Проект создан в рамках выпускной квалификационной работы (ВКР).

## 🚀 Основные возможности

- **Автоматизированный калькулятор**: Мгновенный расчет полной себестоимости ПО на основе трудозатрат, ставок и коэффициентов.
- **Детальная аналитика**: Разбивка затрат по категориям:
  - Фонд оплаты труда (ФОТ)
  - Материальные затраты
  - Накладные расходы
- **Интерактивная визуализация**: Наглядные диаграммы распределения бюджета.
- **Современный стек**: React + TypeScript + Vite + Tailwind CSS.
- **Минималистичный дизайн**: Профессиональный интерфейс, ориентированный на удобство использования.

## 🛠 Технологический стек

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS (Custom Design System)
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Database**: Supabase (подготовлено для интеграции)

## 📐 Методология расчета

Расчет производится на основе классических экономических формул:
- `C = Zт + Zм + Zн`
Где:
- `Zт` — затраты на оплату труда (трудоемкость × ставка)
- `Zм` — материальные затраты (коэффициент 0.1–0.3)
- `Zн` — накладные расходы (коэффициент 0.1–0.25)
