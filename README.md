# SoftCost — Система автоматизированного расчета себестоимости программного обеспечения
 
## Общее описание
SoftCost представляет собой веб-приложение, предназначенное для автоматизации процессов расчета и анализа себестоимости программных продуктов. Разработка выполнена в рамках выпускной квалификационной работы и ориентирована на использование специалистами в области финансового планирования IT-проектов.
 
## Функциональные возможности
- Расчет полной себестоимости ПО на основе вводимых параметров.
- Автоматическое формирование структуры затрат по категориям: фонд оплаты труда, материальные затраты и накладные расходы.
- Визуализация распределения финансовых ресурсов.
- Сохранение параметров расчета для последующего анализа.
 
## Технологический стек 
- Фронтенд-часть: React 18, TypeScript. 
- Стилизация: Tailwind CSS.
- Иконки: Lucide React.  
- Сборка проекта: Vite.
- Интеграция с БД: Supabase. 

## Инструкция по запуску

### Предварительные требования
Для работы проекта необходима среда выполнения Node.js.

### Установка и запуск
1. Установка необходимых зависимостей:
   ```bash
   npm install
   ```
2. Запуск приложения в режиме разработки:
   ```bash
   npm run dev
   ```
3. Сборка оптимизированного пакета для развертывания:
   ```bash
   npm run build
   ```

## Методологическая база
Алгоритм расчета базируется на общепринятых экономических формулах определения себестоимости разработки ПО:
- Общая формула: C = Zт + Zм + Zн
- Затраты на оплату труда (Zт): трудоемкость в человеко-часах, помноженная на часовую ставку.
- Материальные затраты (Zм): определяются через коэффициент в диапазоне 0.1–0.3 от Zт.
- Накладные расходы (Zн): определяются через коэффициент в диапазоне 0.1–0.25 от Zт.
